# `gov-cn` preset

## 适用场景

中文政府公文、正式通知、情况报告、请示类材料的基线样式。

## 当前版式基线

- 页面大小：A4
- 页边距：Word 默认“常规”（上/下 `2.54 cm`，左/右 `3.18 cm`）
- 正文：`仿宋_GB2312`
- 正文字号：三号
- 正文行距：固定值 `28 磅`
- 正文首行：缩进两字符
- 标题：居中，默认使用 `方正小标宋简体`
- 一级标题：默认使用黑体
- 二级标题：默认使用 `楷体_GB2312`
- 三级标题：默认使用 `仿宋_GB2312` 加粗
- 一级、二级、三级标题：自动编号

## 推荐 Markdown 写法

- `#`：公文标题
- 标题下方直接写主送机关或称谓段落
- `##`：一级标题，导出为 `一、XXXX`
- `###`：二级标题，导出为 `（一）XXXX`
- `####`：三级标题，导出为 `1. XXXX`
- 普通段落：正文内容，每段空一行
- `![说明](./image.png)`：插入本地图片，按嵌入式输出；图片所在段落改为单倍行距
- `<!-- pagebreak -->`：需要分页时显式插入

编号由导出器自动补齐，Markdown 里只写标题内容，不要手写 `一、`、`（一）`、`1.`。

## 示例

```md
# 关于开展示例工作的通知

各有关单位：

为统一输出格式，现将有关事项通知如下。

## 工作目标

### 总体要求

#### 任务分工

请各单位提高认识，认真落实。
```

## 当前边界

当前只是“公文样式基线”，还不是完整公文排版规范。以下规则后续可以继续补：

- 发文字号
- 签发人
- 主送机关的专门样式
- 成文日期与落款位置
- 附件说明和抄送规则
